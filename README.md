## üöÄ Installation & Usage

Follow these steps to run the project on your local machine:

1. **Clone or Download the repository**
   Create a folder on your computer and place the project files inside it.

2. **Open your terminal**
   Navigate to the project directory.

3. **Install dependencies**
   Run the following command to install the required packages (`qrcode` and `parcel`):
   ```bash
   npm install
   
4. **Run the application Start the development server using Parcel:**
    ```Bash
    npx parcel index.html

5. **View in Browser Parcel will provide a local URL (usually http://localhost:1234). Open this link in your browser to use the QR Generator.**
